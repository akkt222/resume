<summary>苦労した点</summary>

・全文検索の導入

ユーザーが観たい映画により適切にアクセスできるよう、OpenSearchを用いた全文検索機能を導入しました。検索精度がこれにより大きく向上しました。

・デプロイ戦略の立案・実行

アーリーフェーズかつ少人数体制という状況を踏まえ、事業スピードと品質のバランスを取るデプロイ戦略を設計・実行しました。

・監視基盤の構築

品質の劣化が発生しても、監視やログの基盤が無いために対応が後手に回っている状況でした。Datadogの導入を検討からリリースまで一人で対応し、問題を即座に検知し素早く対応できる状態を構築しました。

全体として、プロダクトの安定性と開発スピードの両立を目指し、運用性・信頼性の強化など幅広く貢献しました。単なる開発作業にとどまらず、スタートアップ特有の状況下での最適な運用設計や機能の早期リリースを意識しながら活動しました。

</details>
